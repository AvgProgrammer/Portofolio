# Lampros Katsaros — Portfolio

Personal portfolio website built with HTML, CSS, JavaScript, and a Node.js backend.

## Pages

- **Home** (`index.html`) — About me section and recent projects overview
- **Projects** (`projects.html`) — Full list of projects

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js (Express)
